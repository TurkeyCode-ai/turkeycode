## How It Works (3 steps)

1. 🦃 **Describe** — one prompt, plain English
2. 🔨 **Build** — automated research, planning, coding, QA, bug fixes
3. 🚀 **Deploy** — `turkey deploy` → live in 60 seconds

---

## What You Get

- Research & architecture planning
- 2-5 phased build with focused coding sessions
- Automated QA: smoke, functional, and visual testing
- Bug detection and fixing (up to 5 cycles)
- Code review
- Git branches, commits, and PRs per phase
- Works with any stack (Next.js, Go, Python, Ruby, Rust, etc.)
